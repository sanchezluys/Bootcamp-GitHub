¡Bienvenido a mi repositorio de apuntes y avances del Bootcamp de GitHub patrocinado por Microsoft y Codigofacilito! 🎉

Aquí encontrarás:
- Notas de cada clase 📚
- Ejercicios y ejemplos de código 💻

## ¿Qué encontrarás aquí?
